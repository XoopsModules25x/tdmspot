README FIRST
-----------------------

This module is a new module for article page management.
Each page can contain: blocks replicable in infinite and your items displayed by category and limited in number, all with permission.

Module Options:
-------------------
- Management page with permission.
- Manage blocks with personal option.
- Management of categories with permission.
- Materials Management:
- Limiting the summary X_BREAK with ().
- Ability to view information on the current drive surname, name, email etc. ..
- Possibility to create a summary X_PAGE with ().
- Ability to view the article X date.
- Upload possible.
- Permission for:
--- See, Proposing, Export (PDF and print), voting, social bar, RSS, Download.


Module Blocks:
-----------------
- Article by alphabetical order, Publication Date, Display, Popularity, Review, Random.
